# Job Application CRM

A lightweight, local-first CRM for tracking job applications.

## Features

- **Kanban Board**: Drag and drop support via status updates.
- **Add Jobs**: Track company, role, salary, and notes.
- **Persistence**: Data saved to LocalStorage.
- **Export/Import**: Backup your data to JSON.

## Getting Started

1.  `npm install`
2.  `npm run dev`

## Built With

-   React + TypeScript + Vite
-   Tailwind CSS
-   Lucide React
